Este es un proyecto de Seguridad para alertar a las personas de una localidad sobre incidentes ocurridos cerca de su posición.
La aplicación tiene la función de mandar una alerta ubicando su posición aproximada y mandando un reporte de incidente ocurrido ya sea del tipo accidente, criminal o emergencia.
Esta hecho para que cualquier emergencia posible cualquier persona cercana con la aplicación instalada pueda saber lo que pasa y pueda socorrer o pedir ayuda a los elementos de emergencia.
